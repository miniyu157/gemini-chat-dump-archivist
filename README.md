# Gemini Chat Dump Archivist

将 Gemini 聊天记录导出为 JSON，并精准保留 Markdown 格式。

---

**特性:**

- 极简、纯粹的逻辑与交互，代码行数 150 左右（LOC）；
- 利用 turndown 还原原始 Markdown，输出纯粹 JSON 结构；
- 提供自动循环滚动到顶部的便捷功能。

**入口:**

- 隐藏在会话的标题处，单击显示菜单
